# 🚀 QUICK START GUIDE: Universal CSV to JSONL Converter v4

## YES! You can absolutely drop it in your datasets directory and auto-convert everything!

## Method 1: Auto-Convert All CSVs in Directory (EASIEST)

### Step 1: Place the script in your datasets directory
```bash
# Copy the converter to your datasets folder
cp csv_converter_v4_universal.py /path/to/your/datasets/
cd /path/to/your/datasets/
```

### Step 2: Run auto-conversion on all CSV files
```bash
# Convert ALL CSV files in current directory
python csv_converter_v4_universal.py . --batch

# This will:
# - Find all *.csv files automatically
# - Detect their format (alpaca, Q&A, text, etc.)
# - Convert each CSV to a matching .jsonl file
# - Show progress for each file
```

### Step 3: Results
```
Your directory structure will look like:
datasets/
├── csv_converter_v4_universal.py
├── math_dataset.csv → math_dataset.jsonl ✅
├── science_data.csv → science_data.jsonl ✅
├── literature.csv → literature.jsonl ✅
└── reasoning.csv → reasoning.jsonl ✅
```

## Method 2: Convert Single Files

```bash
# Convert one specific file
python csv_converter_v4_universal.py my_dataset.csv

# Convert with custom output name
python csv_converter_v4_universal.py data.csv -o training_data.jsonl

# Convert large file with row limit
python csv_converter_v4_universal.py huge_file.csv --max-rows 50000
```

## Method 3: Advanced Batch Options

```bash
# Convert only files matching pattern
python csv_converter_v4_universal.py . --batch --pattern "*_train.csv"

# Convert with specific schema
python csv_converter_v4_universal.py . --batch --schema alpaca

# Convert large datasets with row limits per file
python csv_converter_v4_universal.py . --batch --max-rows 100000
```

## Troubleshooting

### If your CSV has unusual column names:
```bash
# Check what columns it detected
python csv_converter_v4_universal.py your_file.csv

# It will show: "Column mapping: {'instruction': 'your_column_name', ...}"
```

### For custom column mapping:
```bash
# If your CSV has columns: "prompt", "context", "response"
python csv_converter_v4_universal.py data.csv --schema custom --mapping '{"instruction":"prompt", "input":"context", "output":"response"}'
```

### For very large files:
```bash
# Process in chunks with progress updates
python csv_converter_v4_universal.py big_file.csv --chunk-size 5000
```
