## Dataset Specifications

### Format Details
- **File Format**: JSONL (JSON Lines) - one JSON object per line
- **Schema**: Alpaca format with `instruction`, `output`, and optional `input` fields
- **Encoding**: UTF-8
- **File Size**: 31.72 MB (11,086 bytes per sample average)
- **Total Estimated Tokens**: ~3,453,018 tokens

### Sample Structure
```json
{
    "instruction": "Analyze the principles of progressive overload and specificity in designing effective fitness programs",
    "input": "I'm studying for my personal training certification and fitness instruction.",
    "output": "Physical education requires deep understanding of human movement, exercise science, and health promotion to develop lifelong fitness habits and movement skills. Let me guide you through this comprehensive physical education framework..."
}
```

**Required Fields:**
- `instruction`: Practical skills analysis task or life readiness challenge (average 11.4 words, 87 characters)
- `output`: Comprehensive analysis with evidence-based reasoning (average 1,519 words, 10,776 characters)

**Optional Fields:**
- `input`: Context scenario (present in 39.1% of samples, average 80 characters)
