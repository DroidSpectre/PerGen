Dataset Overview
The dataset provides extensive coverage across multiple domains of critical thinking and problem-solving:

Core Categories (1,000 problems total):

Logic & Reasoning Fundamentals (~25 problems)

Deductive, inductive, and abductive reasoning

Logical fallacies and argument analysis

Correlation vs. causation

Valid vs. sound arguments

Problem-Solving Methodologies (~30 problems)

Systematic approaches (PDCA, 5 Whys, OODA loop)

Convergent vs. divergent thinking

Design thinking and systems thinking

Root cause analysis techniques

Logic Puzzles & Brain Teasers (~85 problems)

Classic puzzles (Einstein's riddle, bridge crossing)

Knights and knaves problems

Prisoner dilemmas and game scenarios

Complex deduction problems

Pattern Recognition (~40 problems)

Number sequences (Fibonacci, geometric, arithmetic)

Letter patterns and visual sequences

Rule identification and continuation

Mathematical pattern analysis

Critical Thinking Skills (~30 problems)

Bias identification and mitigation

Source credibility evaluation

Assumption questioning

Evidence assessment

Mathematical Logic (~40 problems)

Probability puzzles and paradoxes

Optimization problems

Statistical reasoning

Logical mathematical relationships

Decision-Making Frameworks (~30 problems)

Cost-benefit analysis

Decision trees and matrices

Risk assessment

Priority setting

Advanced Topics (~720 problems across):

Game Theory - Nash equilibrium, strategic thinking, cooperation

Probability & Statistics - Bayes' theorem, base rates, sampling

Cognitive Biases - Anchoring, confirmation bias, Dunning-Kruger

Systems Thinking - Feedback loops, emergence, leverage points

Spatial Reasoning - 3D visualization, geometric problems

Metacognition - Learning strategies, expertise effects

Complex Riddles - Multi-step lateral thinking problems

Optimization - Algorithmic thinking, complexity analysis

Key Features:

1,000 exactly formatted rows with instruction, input, output structure

Varied instruction types: "What" (496), "Explain" (115), "How" (113), "Describe" (38)

Progressive difficulty from basic concepts to advanced applications

Comprehensive explanations with step-by-step reasoning

Practical applications connecting theory to real-world scenarios

Educational Value:

Develops systematic thinking and analytical skills

Teaches recognition and mitigation of cognitive biases

Builds problem decomposition and solution strategies

Enhances logical reasoning and argument evaluation

Provides frameworks for complex decision-making

Cultivates metacognitive awareness and learning strategies

The dataset emphasizes deep understanding over memorization, providing detailed explanations that help learners understand not just what to think, but how to think more effectively across diverse problem domains.
