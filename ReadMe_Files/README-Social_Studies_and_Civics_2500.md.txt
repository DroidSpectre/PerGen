## Dataset Specifications

### Format Details
- **File Format**: JSONL (JSON Lines) - one JSON object per line
- **Schema**: Alpaca format with `instruction`, `output`, and optional `input` fields
- **Encoding**: UTF-8
- **File Size**: 18.94 MB (7,620 bytes per sample average)
- **Total Estimated Tokens**: ~1,931,428 tokens

### Sample Structure
```json
{
    "instruction": "Analyze the causes of the American Revolution using primary source documents and economic data",
    "input": "I'm preparing for AP History and need to understand complex historical connections.",
    "output": "Historical analysis requires systematic examination of evidence, multiple perspectives, and understanding of cause-and-effect relationships over time. Let me guide you through this comprehensive analytical framework..."
}
```

**Required Fields:**
- `instruction`: Social studies analysis task or historical question (average 10.7 words, 81 characters)
- `output`: Comprehensive analysis with critical reasoning (average 1,015 words, 7,671 characters)

**Optional Fields:**
- `input`: Context scenario (present in 38.2% of samples, average 80 characters)
