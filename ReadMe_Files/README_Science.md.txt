Dataset Overview
The science dataset covers major scientific domains with balanced distribution:

Core Scientific Areas:

Physics (~82 problems) - Mechanics, thermodynamics, waves, electricity, magnetism, modern physics

Chemistry (~84 problems) - Atomic structure, chemical reactions, periodic table, stoichiometry, bonding

Biology (~77 problems) - Cell biology, genetics, evolution, ecology, human body systems

Earth Science (~28 problems) - Geology, weather, climate, environmental science

Astronomy (~21 problems) - Solar system, stars, galaxies, space exploration, cosmology

Additional Topics:

Scientific method and experimental design

Laboratory techniques and equipment

Health science and human physiology

Technology applications and materials science

Environmental issues and sustainability

Mathematical calculations in science contexts

Key Features:

500 exactly formatted rows with instruction, input, output columns

Varied instruction types: "What" questions (316), calculations (50), explanations (31), "How" questions (20)

Multiple difficulty levels from basic concepts to advanced calculations

Factually accurate content with proper scientific notation and units

Step-by-step solutions for calculation problems

Comprehensive explanations for conceptual questions

Content Quality:

All scientific facts are accurate and up-to-date

Includes proper scientific terminology and notation

Provides complete explanations suitable for educational use

Covers both theoretical knowledge and practical applications

Follows educational standards for science instruction

The dataset uses empty input fields since most scientific questions are self-contained. Each output provides detailed, educational responses that would help train language models to understand and explain scientific concepts effectively.
