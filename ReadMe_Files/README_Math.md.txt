Dataset Overview
The dataset includes a wide variety of mathematical topics:

Core Areas Covered:

Basic Arithmetic - Addition, subtraction, multiplication, division

Algebra - Linear equations, quadratic equations, factoring, systems of equations

Geometry - Area, perimeter, volume calculations, coordinate geometry

Calculus - Derivatives, integrals, limits, critical points

Statistics - Mean, median, mode, range, standard deviation

Trigonometry - Basic trig functions, angle conversions

Number Theory - Prime numbers, GCD, LCM, factorization

Fractions & Percentages - Operations with fractions, percentage calculations

Word Problems - Real-world applications across various contexts

Probability - Basic probability, combinations, experimental probability

Unit Conversions - Length, time, weight conversions

Sequences - Arithmetic sequences, pattern recognition

Mathematical Concepts - Definitions and explanations

Key Features:

500 exactly formatted rows with columns: instruction, input, output

Varied difficulty levels from elementary to advanced

Clear, step-by-step solutions showing work when appropriate

Diverse instruction types (solve, calculate, find, explain, convert, etc.)

Factually accurate content with proper mathematical notation

Educational format suitable for language model training

The dataset uses an empty input field for most problems since the mathematical questions are self-contained in the instruction field. Each output provides complete, detailed solutions that would help a language model learn to solve similar problems effectively.
