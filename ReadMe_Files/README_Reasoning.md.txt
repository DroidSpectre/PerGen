Dataset Overview
The reasoning dataset covers fundamental thinking and problem-solving skills across multiple dimensions:

Core Reasoning Areas:

Logical Reasoning (~272 problems) - Deductive, inductive, and abductive reasoning; syllogisms; logical structure

Critical Thinking (~238 problems) - Argument evaluation, assumption identification, evidence assessment

Pattern Recognition (~139 problems) - Sequences, mathematical patterns, visual patterns, rule identification

Decision Making (~107 problems) - Uncertainty handling, risk assessment, priority setting, choice evaluation

Fallacies (~61 problems) - Common reasoning errors, informal fallacies, argument weaknesses

Problem Solving (~34 problems) - Systematic approaches, methodologies, strategy selection

Advanced Topics:

Cognitive Biases - Confirmation bias, anchoring, availability heuristic, overconfidence

Metacognition - Thinking about thinking, self-awareness, reasoning monitoring

Creative Reasoning - Lateral thinking, analogical reasoning, innovation processes

Domain-Specific Reasoning - Legal, scientific, statistical, moral, economic reasoning

Abstract Thinking - Conceptual reasoning, generalization, theoretical understanding

Content Features:

500 exactly formatted rows with instruction, input, output structure

Varied instruction types: "What" questions (364), "Find/Complete" tasks (36), "How" explanations (28)

Multiple difficulty levels from basic definitions to complex analysis

Practical applications with real-world scenarios and examples

Step-by-step explanations for logical processes and problem-solving methods

Educational Value:

Develops critical thinking skills essential for academic and professional success

Teaches identification and avoidance of common reasoning errors

Provides frameworks for systematic problem-solving and decision-making

Enhances pattern recognition and analytical abilities

Builds metacognitive awareness for improved learning and thinking

The dataset uses empty input fields since reasoning questions are typically self-contained. Each output provides comprehensive explanations that help learners understand not just what to think, but how to think more effectively across diverse contexts.
