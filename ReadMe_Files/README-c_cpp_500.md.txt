# C/C++ Dataset (500 samples)

## Overview
This dataset includes 500 C and C++ code examples covering foundational through advanced system programming and optimization techniques.

## Categories (50 samples each)
1. Basic Syntax & Fundamentals
2. Pointers & Memory Management
3. Data Structures & Arrays
4. Functions & Scope
5. Object-Oriented Programming (C++)
6. STL & Standard Library (C++)
7. File I/O & Streams
8. Error Handling & Debugging
9. System Programming
10. Performance & Optimization

## File Structure
- `instruction`: Task description
- `input`: Provided input (if any)
- `output`: Source code snippet

## Usage
1. Load `c_cpp_500.csv` in training pipeline.
2. Compile with `g++ -std=c++11` or `gcc` and run to validate.

## Prerequisites
- GCC or Clang supporting C++11+
- Standard C library
- Linux or Windows development environment

## Testing
- All samples compile without warnings using `-Wall` and run correctly.

## Integration
Use for fine-tuning models on C/C++ code generation, system-level programming, and performance optimization tasks.
