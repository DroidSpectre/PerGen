# Python Dataset (500 samples)

## Overview
This dataset contains 500 Python code examples spanning basic syntax to advanced scripting, suitable for Python 3.6+ environments.

## Categories (50 samples each)
1. Basic Syntax & Fundamentals
2. Data Structures & Collections
3. Functions & Modules
4. Object-Oriented Programming
5. File I/O & Data Processing
6. Error Handling & Debugging
7. Libraries & Frameworks
8. Web Development & APIs
9. Data Science & Analytics
10. Automation & Scripting

## File Structure
- `instruction`: Task description
- `input`: Provided input (if any)
- `output`: Python code snippet

## Usage
1. Load `python_500.csv` into training pipeline.
2. Use `instruction` to prompt model and validate `output` code in a Python REPL.

## Prerequisites
- Python 3.6 or later
- Libraries: requests, flask, pandas, numpy, flask

## Testing
- All samples executed without syntax errors in a virtual environment.

## Integration
Ideal for fine-tuning models on Python code generation, explanation, and debugging tasks.
