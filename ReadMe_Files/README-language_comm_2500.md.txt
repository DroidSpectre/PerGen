## Dataset Specifications

### Format Details
- **File Format**: JSONL (JSON Lines) - one JSON object per line
- **Schema**: Alpaca format with `instruction`, `output`, and optional `input` fields
- **Encoding**: UTF-8
- **File Size**: 20.52 MB (8,026 bytes per sample average)
- **Total Estimated Tokens**: ~1,917,879 tokens

### Sample Structure
```json
{
    "instruction": "Analyze how conversational structure affects meaning-making and relationship building",
    "input": "I'm preparing for advanced composition courses and need to improve my writing sophistication.",
    "output": "Conversation analysis provides systematic tools for understanding how people create meaning and manage relationships through talk-in-interaction. Let me guide you through this interactional framework..."
}
```

**Required Fields:**
- `instruction`: Language/communication task or analysis (average 9.5 words, 80 characters)
- `output`: Comprehensive analysis with linguistic reasoning (average 1,009 words, 8,319 characters)

**Optional Fields:**
- `input`: Context scenario (present in 39.2% of samples, average 78 characters)
